## Padawan Practice Drills

### Drill 1: Name That Corner

**Goal:** Learn to identify corner types by feel

**Process:**
1. Drive one lap at Winton (or your current track)
2. After each corner, name the type:
   - Hairpin? Carousel? Sweeper? 90-degree? Kink?
3. Check your answers using track guide

**Success:** 80%+ correct identification

### Drill 2: The Apex Experiment

**Goal:** Feel the difference between early and late apex

**Process:**
1. Pick ONE corner (any track)
2. **Lap 1:** Early apex (apex 1/3 into corner)
3. **Lap 2:** Late apex (apex 2/3 into corner)
4. **Compare:** Exit speed (which is faster?)

**Success:** Late apex is faster (almost always)

### Drill 3: Track Walk (Mental)

**Goal:** Describe track using proper terminology

**Process:**
1. Pull up track map (Winton, Rudskogen, Jefferson)
2. Write description of each corner using proper terms:
   - "T1: Right sweeper, 65 mph entry"
   - "T2: Hairpin, late apex, 2nd gear"
3. Practice until it flows naturally

**Success:** Can describe any track clearly and accurately
