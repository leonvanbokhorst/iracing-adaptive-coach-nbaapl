**Purpose:**

Chicanes are artificial. Track designers add them to break up long straights (for safety) or to increase technical difficulty.

**Technique:**

1. **Brake once** for the whole sequence
2. **Compromise the first apex** to set up the second
3. **Roll speed through** (don't stop and start)
4. **Exit speed from the last corner** is priority

**Why This Works:**

If you treat a chicane as two separate corners, you'll brake twiceâ€”once for each corner. That kills momentum and wastes time.

Instead: **One brake zone, flowing through both corners, optimize the exit.**

**The Compromise:**

- First corner: Slightly wider/earlier apex than ideal
- Second corner: Perfect apex for maximum exit speed

**Ray FF1600 Tip:**

Maintain flow. Two separate brake zones = you're over-slowing.

**Examples:**

- **Jefferson T1-T2:** Some tracks have chicanes (not Jefferson specifically, but common)
- **Winton T1-T2:** Could be treated as chicane-like (sweeper into hairpin)

**Technique:**

1. **Sacrifice the first corner** to set up the second
2. **Quick steering reversal** between corners
3. **Focus on exit** of the final corner
4. **Weight transfer management** critical

**Why It's Different from Esses:**

Esses are flowing. Switchbacks are **abrupt**. You're making sharp direction changes, not gentle curves.

**The Challenge:**

Weight transfer happens fast. Car loads outside tires in first corner, then immediately loads opposite side in second corner. The car never settles.

**Solution:** Smooth inputs, progressive steering reversal, patience on throttle.

**Examples:**

- **Winton T4:** Left Switchback after T3
- **Rudskogen:** Several in technical sections

**Ray FF1600 Tip:**

Don't fight the car. Let it transfer weight naturally between corners.

**Master Lonn's Challenge:**

Switchbacks feel chaotic. They are. Embrace the chaos, focus on the exit.

## Corner Combination Principles

### Universal Rules

**1. Treat sequences as ONE corner**
- Don't brake multiple times
- Flow > peak speed in any single element
- Optimize the LAST corner's exit

**2. Compromise early, perfect late**
- First corner: Good enough to set up the next
- Middle corners: Maintain rhythm
- Last corner: Maximum exit speed

**3. Weight transfer is critical**
- Quick direction changes = rapid weight shifts
- Smooth inputs prevent oscillations
- Let the car settle between changes

**4. The exit always wins**
- If in doubt, sacrifice entry for exit
- The last corner's exit determines your straight-line speed
- Exit speed compounds (carries forward)
